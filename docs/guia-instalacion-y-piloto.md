# Guia de instalacion y piloto FeriAPP

Esta guia sirve para explicar a las primeras usuarias como instalar FeriAPP en el telefono y que probar durante el piloto.

## Enlace oficial

Usar siempre la URL publicada en Vercel:

```text
https://feri-app-gamma.vercel.app
```

No usar enlaces `file://` para pruebas en telefono. La instalacion PWA necesita una URL web segura con `https`.

## Como instalar en Android

Recomendado: Google Chrome.

1. Abrir Chrome en el telefono.
2. Entrar a `https://feri-app-gamma.vercel.app`.
3. Esperar que cargue FeriAPP.
4. Si aparece un aviso, tocar `Instalar app`.
5. Si no aparece aviso, tocar el menu de tres puntos de Chrome.
6. Elegir `Instalar app` o `Agregar a pantalla principal`.
7. Confirmar con `Instalar` o `Agregar`.
8. Buscar el icono de FeriAPP en la pantalla de inicio.

Texto simple para usuarias:

```text
En Android abre este enlace con Chrome:
https://feri-app-gamma.vercel.app

Luego toca los tres puntos de arriba, elige "Instalar app" o "Agregar a pantalla principal" y confirma. Despues podras abrir FeriAPP desde el icono del telefono.
```

## Como instalar en iPhone

Recomendado: Safari.

1. Abrir Safari en el iPhone.
2. Entrar a `https://feri-app-gamma.vercel.app`.
3. Tocar el boton `Compartir`.
4. Elegir `Agregar a pantalla de inicio`.
5. Tocar `Agregar`.
6. Buscar el icono de FeriAPP en la pantalla de inicio.

Texto simple para usuarias:

```text
En iPhone abre este enlace con Safari:
https://feri-app-gamma.vercel.app

Luego toca Compartir, elige "Agregar a pantalla de inicio" y despues "Agregar". Asi FeriAPP queda instalada como icono en tu telefono.
```

## Si no aparece la opcion de instalar

Revisar esto:

1. Que se este usando la URL `https://feri-app-gamma.vercel.app`.
2. Que no se este abriendo desde un archivo local `file://`.
3. En iPhone, usar Safari para explicar el flujo mas facil.
4. En Android, usar Chrome actualizado.
5. No usar modo incognito o privado.
6. Si ya estaba instalada, borrar el icono anterior y volver a instalar despues del nuevo despliegue.
7. Esperar unos segundos en la pantalla inicial antes de abrir el menu del navegador.

## Que debe probar una usuaria piloto

Primera prueba, idealmente en 5 a 10 minutos:

1. Instalar FeriAPP en el telefono.
2. Crear su cuaderno.
3. Agregar 3 productos reales.
4. Anotar una venta con botones grandes.
5. Guardar un resumen del dia.
6. Registrar sobrantes.
7. Abrir Modo Feria y leer recomendaciones.
8. Decir que parte entendio rapido y que parte le confundio.

## Preguntas para recoger feedback

1. Te tomo menos tiempo que anotarlo en papel?
2. Entendiste como agregar productos?
3. Entendiste como registrar una venta?
4. El cuaderno se parece a la forma en que anotas hoy?
5. La recomendacion de Modo Feria te sirvio?
6. Que texto, boton o pantalla te confundio?
7. Pagarias por FeriAPP Plus si te ayuda a perder menos producto o vender mejor?

## Recomendacion para el piloto

Durante el piloto conviene activar FeriAPP Plus gratis a usuarias invitadas. Asi prueban el valor real del producto: recomendaciones, Modo Feria con cantidades, alertas y resumen semanal.

La version gratis sirve para que puedan entrar sin barrera, pero el aprendizaje fuerte del piloto esta en saber si las recomendaciones Plus ayudan a decidir mejor.
